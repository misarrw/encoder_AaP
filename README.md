# Hystorical Encoder
Ваш инструмент для работы с классическими шифрами: Цезарь, Аффинный, Аффинный рекуррентный, Хилл, Хилл рекуррентный, Виженер (3 вида гаммы), Вернам.

## Описание шифров

### 1. Подстановочные шифры
#### 1.1 Шифр Цезаря

Принцип: Сдвиг каждой буквы алфавита на фиксированное число.
Ключ: Число (сдвиг).
 
#### 1.2 Аффинный шифр

Принцип: Замена буквы по формуле (a * x + b) mod 26.
Ключ: Два числа (a, b).

#### 1.3 Аффинный рекуррентный шифр

Принцип: Аффинный шифр, где ключи (a, b) меняются по рекуррентной формуле.
Ключ: Начальные (a1, b1, a2, b2).

### 2. Блочные шифры
#### 2.1 Шифр Хилла

Принцип: Шифрование блоков букв с помощью матрицы.
Ключ: Матрица 2x2 (например, [1, 2, 3, 4]).

#### 2.2 Рекуррентный шифр Хилла

Принцип: Как Хилл, но матрица ключа меняется по рекуррентному правилу.

### 3. Шифры гаммирования
#### 3.1 Шифр Виженера

Принцип: Шифрование с использованием ключевого слова и гаммирования.
Типы гаммы:
rep – повторение ключа.
open – гамма на основе открытого текста.
cipher – гамма на основе шифртекста.

#### 3.2 Шифр Вернама

Принцип: Побитовый XOR текста с ключом (одноразовый блокнот).
Ключ: Случайная последовательность той же длины, что и текст.
